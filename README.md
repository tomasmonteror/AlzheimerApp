# AlzheimerApp 🧠

Aplicación Android orientada a estimulación cognitiva en pacientes con Alzheimer y deterioro cognitivo leve/moderado.

---

## 🎯 Objetivos del proyecto

- Favorecer la **estimulación cognitiva** mediante ejercicios interactivos.
- Mantener y reforzar **memoria visual y reconocimiento**.
- Adaptar la dificultad progresivamente según el nivel del usuario.
- Permitir el uso de **imágenes personalizadas** (familiares, objetos cotidianos).
- Proporcionar una herramienta útil tanto para:
  - Uso doméstico (cuidadores/familia)
  - Contexto clínico (terapeutas, neuropsicólogos)

---

## 🧠 Enfoque clínico

El diseño está pensado para:

- Pacientes con:
  - Alzheimer en fases iniciales y avanzadas
  - Deterioro cognitivo leve
- Apoyo en terapias de:
  - Rehabilitación cognitiva
  - Estimulación de memoria episódica y semántica
- Uso de **refuerzos positivos** (feedback visual)
- Interfaz:
  - Sencilla
  - Accesible
  - Minimiza carga cognitiva

---

## 🎮 Tipos de juegos

### 1. Juego de reconocimiento

- Las cartas aparecen **boca arriba**.
- El usuario debe seleccionar las parejas correctas.
- Diseñado para:
  - Pacientes en fases más avanzadas
- Reduce carga de memoria, favoreciendo:
  - Asociación visual
  - Identificación

---

### 2. Juego de emparejamiento (Memoria)

- El usuario debe encontrar parejas de cartas iguales.
- Adaptación progresiva:
  - Incremento del número de cartas por nivel
  - Inclusión de imágenes + elementos genéricos (ej: emojis)
- Entrena:
  - Memoria de trabajo
  - Atención
  - Reconocimiento visual

---

## 🖼️ Uso de imágenes personalizadas

- Permite cargar:
  - Fotos de familiares
  - Objetos cotidianos
  - Elementos significativos para el paciente

- Objetivo:
  - Reforzar memoria emocional
  - Mejorar la conexión con el entorno
  - Aumentar la motivación del usuario

---

## ⚙️ Gestión de imágenes

La app incluye un sistema de:

- Añadir imágenes desde galería
- Clasificar en:
  - Objetos
  - Recompensas
- Reordenar imágenes:
  - Drag & drop
  - Botones de control (↑ ↓)
- Eliminar imágenes

## 🔧 Tecnologías utilizadas

- Kotlin
- Jetpack Compose
- Android SDK
- Coil (carga de imágenes)
- Reorderable LazyLists
- Persistencia de datos mediante almacenamiento local
